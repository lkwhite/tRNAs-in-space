# Contributing

This is primarily a resource repository for sharing pre-computed tRNA coordinate datasets.

## Using the Tool

Feel free to:
- Use the pre-computed datasets
- Fork the repository for your own datasets
- Run the tool on your own sequences
- Open issues for bugs or questions

## Bug Reports

Found a bug? [Open an issue](https://github.com/lkwhite/tRNAs-in-space/issues) with what you tried and what happened.

## Questions

Have a question? Check the [FAQ](FAQ.md) or [open an issue](https://github.com/lkwhite/tRNAs-in-space/issues).

## Forking

Want to create your own dataset collection? Fork this repository and customize it for your needs!
